# ☁️ CloudVault

CloudVault is a modern cloud-based file storage platform inspired by Google Drive.  
It allows users to securely upload, manage, and delete files using AWS S3 cloud storage with Google OAuth authentication.

---

# ⚙️ Installation Guide

## 1️⃣ Clone Repository

```bash
git clone https://github.com/JATINJAYASIMHA/cloudvault.git
```

---

## 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

---

## 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```

---

# 🔐 Environment Variables

Create `.env` inside backend folder:

```env
PORT=5000

MONGO_URI=your_mongodb_uri

AWS_ACCESS_KEY=your_access_key
AWS_SECRET_KEY=your_secret_key
AWS_REGION=ap-south-1
AWS_BUCKET_NAME=your_bucket_name
```
